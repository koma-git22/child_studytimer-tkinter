# child_studytimer
# ⏰ 親子スケジュール応援アプリ (Morning Schedule Timer)

お子様の「あさのじゅんび」を楽しく、自発的に進められるように開発した、スマホ風デザインの知育タイマーアプリです。
親子の朝の時間を笑顔にし、子どもが「できたー！！」という達成感を味わえる工夫がたくさん詰まっています。

## ✨ 主な特徴 (Features)

- **子どものやる気を引き出すデザイン＆カラー**
  - アプリ全体に優しい薄ピンクを採用。
  - 「できたー！！」ボタンやピンチ時のアラートには、「イチゴみるくレッド」を使用しています。
- **直感的にわかる手作りの角丸プログレスバー**
  - OSに依存しないカプセル型の可愛いプログレスバーをCanvasで自作。残り時間が10%以下になると自動で水色から可愛い赤色に変化し、視覚的に応援します。
- **選べる3つの時間設定**
  - 忙しい朝でもワンタップで選べる「5分」「20分」「30分」の選択ボタンを搭載。どれを選んでいるかも色でパッと分かります。
- **大喜びする2回ジャンプ＆ご褒美スタンプ**
  - 準備が終わってボタンを押すと、お気に入りのキャラクター（`pan.jpg`）が嬉しそうに2回ぴょんぴょんと跳ね、画面に「⭐⭐⭐」のスタンプが出現します。
- **状況に合わせた優しいメッセージ分岐**
  - 時間内にできたら「やったね！よくできました！」、もし時間切れ（00:00）になっても、最後まで準備を終えてボタンを押せば「さいごまで がんばったね！」と優しく褒めて次のやる気に繋げます。

## 🛠 技術的なこだわり (Technical Highlights)

- **マルチスレッドによるフリーズ防止 (`threading` の活用)**
  - タイマーのカウントダウン処理をメイン画面とは別のスレッド（裏方）で動かしているため、タイマー作動中も画面が固まらず、ボタン操作がスムーズに行えます。
- **徹底したフlying・誤操作防止ガード**
  - アプリ起動直後に「できたボタン」を押してしまうフライングを防止。
  - すでに成功画面になっている時にスタートボタンを連打してもタイマーが多重起動しないよう、強固な安全ガードをロジックに組み込んでいます。
- **Mac/Windows両対応の角丸・カーソル形状**
  - ボタンがボタンらしく見えるよう、マウスホバー時に指マーク (`cursor="hand2"`) に変わる仕様をキャンバスボタンに適用しています。

## 💻 必要環境 (Requirements)

### 1. 動作環境 (OS)
- **Mac** (macOS 10.15以上推奨) 
- **Windows** (Windows 10 / 11)

### 2. 必須言語 (Language)
- **Python 3.8 以上** (Python 3.14.6 動作確認済み)

### 3. 必要外部ライブラリ (Python Packages)
JPG画像を読み込み・リサイズするために **Pillow** ライブラリが必要です。

```bash
pip install Pillow
```

### 4. 必要なファイル配置 (Assets)
アプリを起動する前に、以下の画像ファイルをご自身のパソコンの指定場所に配置してください。
- **配置パス:** `/Users/kitamuramaho/Downloads/pan.jpg`
- **ファイル名:** `pan.jpg`

## 🚀 使い方 (Usage)
1. リポジトリからコードをダウンロードします。
2. 上記の必要環境（Pillowのインストール、画像の配置）を整えます。
3. ターミナルまたはコマンドプロンプトでプログラムを実行します。

```bash
python schedule_app.py
```

4. 上部のボタン（5分 / 20分 / 30分）から好きな時間を選びます。
5. 「よーい、スタート！」ボタンを押すとカウントダウンが始まります。（途中で一時停止・再開も可能です）
6. 準備ができたら、大きな「できたーー！！」ボタンを押してパンダと一緒に喜びましょう！
