# ⏰ 親子スケジュール応援アプリ (あさのじゅんび)

お子様の朝の準備やタスクを楽しく応援するための、Python（Tkinter）製のカウントダウンタイマーアプリです。かわいいデザインと、時間切れ前の警告カラー、達成時のキャラクターアニメーションやスタンプ機能で、子どものやる気を引き出します。

## ✨ 主な機能

* **選べるタイマー設定:** ワンタップで「5分」「20分」「30分」の目標時間を切り替え可能。
* **視覚的なプログレスバー:** 手作りの角丸ゲージで、残り時間がひと目で分かります。
* **安心の警告演出:** 残り時間が10%以下になると、タイマーとゲージがピンクに変化してお知らせ。
* **ご褒美アニメーション:** 準備が終わって「できたー！！」ボタンを押すと、お気に入りのキャラクターがジャンプし、ご褒美スタンプ（⭐⭐⭐）が出現します。
* **選べるリザルト:** 時間内にできた場合（やったね！）と、時間切れ後に完了した場合（がんばったね）で演出が変わります。

---

## 📂 必要なファイル配置（重要）

本プログラムを実行する際は、Pythonスクリプトと同じフォルダ内に画像ファイルを配置してください。

### 📌 推奨するディレクトリ構成
```text
your-project-folder/
├── schedule_app.py  (このPythonプログラムファイル)
└── pan.jpg          (または pan.JPG)
```

> 💡 **仕様:** 実行中のスクリプトの位置を自動検出して `pan.jpg`（または大文字の `.JPG`）を読み込みます。画像ファイルがない場合は `FileNotFoundError` が発生するため、必ず事前に配置してください。

### 🖼️ 配置する画像（pan.jpg）の推奨仕様
アプリのレイアウトを美しく保つため、以下の画像サイズを推奨します。
* **推奨サイズ:** 横 **300 px** × 縦 **100 px** 程度
* **自動リサイズ:** プログラム内で自動的に縮小処理（アスペクト比維持）が行われますが、極端に縦長・横長の画像は表示が崩れる原因になります。

## 🚀 導入・実行方法

### 1. 必要環境のインストール
このアプリはPythonの標準ライブラリ（Tkinterなど）以外に、画像処理ライブラリ **Pillow (PIL)** を使用しています。
事前に以下のコマンドを実行してインストールしてください。

```bash
pip3 install Pillow
```

### 2. アプリの起動
画像（`pan.jpg`）を配置したフォルダ内で、以下のコマンドを実行します。

```bash
python3 schedule_app.py
```

## 🛠️ 使い方

1. **時間を選ぶ:** 画面上部の「5分」「20分」「30分」から目標時間を選択します（デフォルトは20分）。
2. **スタート:** 「よーい、スタート！」ボタンを押すとカウントダウンが始まります。途中で一時停止や再開も可能です。
3. **できた！:** 準備が完了したら、大きなピンクの「できたー！！」ボタンを押します。
4. **お祝い:** キャラクターがピョコピョコ跳ねて、お星さまスタンプがもらえます！
